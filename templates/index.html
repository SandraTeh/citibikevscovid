First Wave and Second Wave Line Graphs



Graph above show average covid-19 cases for the past 7 days and average no of of trip for the past 7 days from March 2020 to June 2020. As we can see from the graph when the cases gone up, no of trip gone down. Vice versa, when the cases decrease, the no of trip gone up. In the month of April which the case number reached the peak, the no of of trip reached the lowest point as well. The number of trips increases as the number of cases decreases after the cases reached the peak in April.



Analysing second wave which started in December 20 to January 21, we can see the pattern is slightly different from first wave. As we can see in the month of January 21, eventhough the cases gone up, the no of trip also haven't gone down. Although there is a big drop

First Wave and Second Wave Model



Using Scikit learn to perform machine learning about the first wave and second wave, we can see from the Support Vector Regression graph above, the linear model is best fit as the data is very close to the model. Analysing the linear regression, the training score is 0.7805 and the testing score is 0.6973 for first wave. For the second wave, the training score is onl 0.6742 and the testing score is only 0.67649. The first wave score is higher than second wave. This again prove that the covid-19 in first wave have got bigger impact in Citibike compared to second wave. In the prediction tool, it is predicting what would be the number of trip if there is third wave. The prediction tool is using the second wave model linear model to predict the number of trips as it will be more realistic to use the latest data. The prediction tool reduced by is comparing the predicted trips to last year average number of trips between Oct 19 to Jan 20.
